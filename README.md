# odonto-rio

🦷 OdontoRio - 

Este projeto é um site institucional moderno desenvolvido para uma clínica fictícia chamada "OdontoRio". O objetivo principal foi criar uma interface limpa, profissional e acolhedora, utilizando práticas de design que transmitem credibilidade e facilitam o agendamento de consultas.

🚀 Funcionalidades e Destaques:

Design Clean e Moderno: Uso estratégico de tipografia (Roboto) e cores que remetem à saúde e higiene.

Seção Hero Impactante: Banner principal com chamada para ação (CTA) clara e direta.

Layout Responsivo: Desenvolvido para oferecer uma excelente experiência de navegação em diferentes tamanhos de tela.

Foco em Conversão: Botões de "Agende sua consulta" estrategicamente posicionados para guiar o usuário.

🛠️ Tecnologias Utilizadas:

HTML: Estruturação semântica do conteúdo.

CSS: Estilização avançada, utilizando Flexbox para o alinhamento de containers e posicionamento de elementos.

Figma: Utilizado como base para a fidelidade visual e extração de dimensões precisas.
